# data/

Place your input video files here before running the scripts.

## Required Files

| File | Used by | Description |
|------|---------|-------------|
| `ALS - Trim20s.mp4` | `optical_flow_als_ultrasound.py` | Real ALS muscle ultrasound video (~20s clip) |
| `fasciculation_simulation_accel_decel_test_new4.mp4` | `optical_flow_synthetic.py` | Synthetic simulation video from `Fasciculation-Simulation` repo |

## Getting the Synthetic Video

The synthetic video is generated by the **Fasciculation-Simulation** repository:
`https://github.com/ShzdiTrnum/Fasciculation-Simulation`

Run the simulation script and use the output `.mp4` as input here.

> **Note:** Video files are excluded from version control via `.gitignore` due to file size.
> Store them locally in this folder and update the path variables in the scripts.
